## White Lake 0.1.0

![White Lake Theme](/screenshot.png?raw=true "Screenshot")

A minimalistic [Ghost](https://github.com/tryghost/ghost/) theme similar to [octopress' White Lake theme](https://github.com/gehaxelt/CSS-WhiteLake).

### Features

* Author & Tag pages
* Custom error page
* Blog navigation
* Post excerpts
* Built using PureCSS
* No Javascript
